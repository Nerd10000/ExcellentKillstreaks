# ExcellentKillStreaks
ExcellentKillStreaks (ExStreaks) is a lightweight but featureful paper plugin for you Minecraft server. It offers a variety of useful features for a killstreaks plugin.The plugin was made for KitPVP or FFA servers.

### 📚 Features
- 📦 **Persistent data** — The data is saved on the disk so you don't need to worry about stopping the server or the host crashing and players lose their killstreaks.
- 🪶 **Light as a feather** — The plugin does not store a lot of stuff in memory and mainly everything is saved on the disk.
- 📂 **Open source** — The plugin's sorce code is available on Github so you can suggest improvements, report issues/bugs. **❗WARNING! THE PLUGIN IS LICENSED UNDER GNU GENERAL PUBLIC LICENSE V3!**
- 🆘 **Fast support** — I am available frequently so send me a friend request and message me if you have any questions!
